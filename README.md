
# Spaceship Transporter

Application that helps to keep track of ships travelling between planets.


## Tech Stack

**Client:** HTML, CSS, JavaScript, Bootstrap, jQuery

**Server:** Java, Spring Boot, Thymeleaf

**Others:** MySQL, Gradle
